# DSA-leetcode-solutions


This repository contains my **C++ solutions to LeetCode problems**, maintained primarily for **learning and practicing Data Structures and Algorithms (DSA)**.

##  Purpose

- Practice **problem-solving** using C++
- Track my learning progress over time
- Revise and improve solutions as my understanding grows

##  Repository Structure

- Each file corresponds to **one LeetCode problem**
- **Filename format**:
<problem-number>-<problem-name>.cpp

**Example:**

- The problem number matches the **official LeetCode question number**.
